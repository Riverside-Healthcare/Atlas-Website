it's called Atlas

Bringing all your reporting together!

A business intelligence library and documentation database.

ETL processes collect metadata from various reporting platforms
and store it in a centrally located database.

A modern web UI is used to add additional documentation to the reports
and also provides an intuitive way to search, favorite and share
reporting content.

Why Atlas?

  • Do you need better governance over your report documentation?
  • Do you want to make that documentation available to end users?
  • How about being able to track and manage what reports end users are using?
  • Being able to define specific concepts metrics that end users could access?

what about...
  • Easily favorite and share reports with coligues?
  • Should your reports be organized into projects?
  • With recurring maintenance tasks?
  • Wait, did you want to add a maintenance schedule to your report?

Atlas does all this and more!

What else does it do?

Atlas is a web user interface for an ETL process that collects report metadata from a variaty of sources. The webapp provides a clean search, a multitude of documentation options, and in a word, makes managing your report library stress free.

Key Features

  • Clearly define terms and link to reports
  • Easily group reports and terms into projects
  • Create and track Data Initiatives
  • Ability to run and open editors directly from Atlas
  • Atlas allows very expansive documentation! - Screenshots - Large editor form with multiple sections - Markdown support - Maintenance schedules and log workflow - Tiered report validation levels - Report relationships are fully ETL'd - Report/term/project/initiative relationships can be added - Source report SQL is included in ETL's
  • Enterprise Data Glossary - custom term definitions can be added and linked to content
  • Usability - Fully searchable - users can find all publicly available content - Native favoriting and sharing - Webapp is embeddable and works in common browsers - Request access forms - Single sign on for all system users
  • Excellent Search - Full Text Index on most fields - SQL is searchable - Search filters
  • Reports, editors and record viewers can be launched directly from Atlas
  • Recommended reports based on users interests

Is there a demo?

Yes!

The demo runs with docker. You can run it on Play With Docker, or locally.

docker run -i -t -p 1234:1234 -e PORT=1234  -u 0 christopherpickering/rmc-atlas-demo:latest

For more tips on running a demo please take a look in the docs.
