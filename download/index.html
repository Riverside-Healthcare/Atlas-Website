how to get Atlas

Open Source

Atlas is an open source project and free to use! There are two parts to Atlas - a webapp and an ETL. The ETL process collects the report metadata which is displayed in the webapp.

Atlas is most easily installed on an IIS windows server .

The ETL is most easily installed on a Microsoft SQL Server .

There is a community of Atlas users who can help you get started!
